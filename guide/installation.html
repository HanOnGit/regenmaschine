Installation¶

Using pip¶

To install Regenmaschine, enter the following in any terminal application:

$ pip install regenmaschine

If you don’t have pip installed, use `the official installation guide <http://docs.python-guide.org/en/latest/starting/installation/`_.

Via the Source Code¶

Regenmaschine’s up-to-date source code can be found on Github: https://github.com/bachya/regenmaschine

Users with git available on their system can clone the repository easily:

$ git clone https://github.com/bachya/regenmaschine.git

Alternatively, tarballs and “zipballs” of the latest source can be downloaded:

$ curl -OL https://github.com/bachya/regenmaschine/tarball/master
$ curl -OL https://github.com/bachya/regenmaschine/zipball/master

With either method, once the source has been downloaded, you can include it in your packages or install it locally:

$ cd regenmaschine/
$ pip install .
